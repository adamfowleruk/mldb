Class: searchresults

searchresults

new searchresults(container)

Displays a list of search results, rendered appropriately for the content. Supports custom renderers. These could render content in the search result or use the URI to fetch more information on the document. Supports both JSON and XML (likely XHTML) content display by default. Also allows making the entire result clickable, to navigate to another application page, supporting dynamic URL creation.

This widget is partially MLJS Workplace enabled.

Parameters:
Name Type Description
container string

HTML ID of the element in which to draw this widget's content
